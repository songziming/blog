# 静态博客解决方案

使用 Python 独立开发的静态博客生成工具

搜索markdown文件，调用pandoc转换为html，使用jinja2渲染成页面

### 为何自己编写生成器

- 静态网站生成工具其实很简单
- 无需实现功能完备的SSG，满足我们的需求即可
- 我们有一些特殊定制要求，例如内嵌diagram、渲染notebook

### 笔记和博客的关系

笔记是非正式的/不成熟的/不完整的想法，有效性也是短期的。
一旦完善，便可以整理为文档，收录在个人博客中，用于长期参考。

笔记需要经常编辑修改，笔记工具更看重编辑体验（Notion）。
博客不经常编辑，经常浏览，更看重阅读体验和查找搜索功能。
