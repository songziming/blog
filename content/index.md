本站是MkDocs生成的静态网站：

- 安装：`python -m pip install mkdocs`
- 构建：`python -m mkdocs build`
- 预览：`python -m mkdocs serve`

使用Typora编写内容，使用[pangu.js](https://github.com/vinta/pangu.js)自动在中英文之间添加空格，使用KaTeX渲染数学公式。不依赖CDN，可以在离线环境下使用。
